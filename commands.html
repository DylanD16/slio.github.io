Information:

Below are all of Slio's commands listed in a nice clean view. I hope this may make a bit more sense than the ?help command.

Commands:

?kick | Kicks a user from the server | ?kick [user] [reason]

---------------------------------------------------------------------------------------------------

?ban | Bans a user from the server | ?ban [user] [reason]

---------------------------------------------------------------------------------------------------

?mute | Assigns a the user the Muted role which stops them from talking | ?mute [user] [reason]

---------------------------------------------------------------------------------------------------

?invite | Invite the bot to your server

---------------------------------------------------------------------------------------------------

?help | DMs you a list of commands to use

---------------------------------------------------------------------------------------------------

?avatar | Gets a users avatar | ?avatar [user]

---------------------------------------------------------------------------------------------------

?suggest | Suggest a feature and vote on it! | ?suggest [Suggestion]
